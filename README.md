# astrbot_plugin_color_converter
颜色格式转换器，支持RGB、CMYK、16进制互转

【命令格式】color <目标格式> <现有格式>

» 示例: color rgb 72C0FF

» 示例: color cmyk 114,166,255

» 示例: color hex 55,35,0,0
            
【参数说明】
<目标格式>: 想要转换成的格式，可选值: rgb, hex, cmyk

<现有格式>: 现有的颜色值，支持以下格式:

- 16进制: 3位或6位16进制数
示例: F00 或 FF0000

- RGB: 3个0-255的数字，用逗号或空格分隔
示例: 255,0,0 或 255 0 0

- CMYK: 4个0-100的数字，用逗号或空格分隔
示例: 0,100,100,0
            
【帮助命令】colorhelp：显示此帮助信息
